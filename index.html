MDN - WebRTC: Still photo capture demo

This example demonstrates how to set up a media stream using your built-in webcam, fetch an image from that stream, and create a PNG using that image. Original article: Taking still photos with WebRTC. Original code: https://github.com/mdn/samples-server/tree/master/s/webrtc-capturestill. 2018-09-15 JFS.

Switch to: HTTP or HTTPS (Chrome only allows HTTPS per MDN)
